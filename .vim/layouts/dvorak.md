## Layer 1 (unshifted):
```
 ┌───┬───┬───┬───┬───┬───┬───┬───┬───┬───┬───┬───┬───┬──────┐
 │ ` │ 1 │ 2 │ 3 │ 4 │ 5 │ 6 │ 7 │ 8 │ 9 │ 0 │ [ │ ] │ bksp │
 ├───┴─┬─┴─┬─┴─┬─┴─┬─┴─┬─┴─┬─┴─┬─┴─┬─┴─┬─┴─┬─┴─┬─┴─┬─┴─┬────┤
 │ tab │ ' │ , │ . │ p │ y │ f │ g │ c │ r │ l │ / │ = │  \ │
 ├─────┴─┬─┴─┬─┴─┬─┴─┬─┴─┬─┴─┬─┴─┬─┴─┬─┴─┬─┴─┬─┴─┬─┴─┬─┴────┤
 │ caps  │ a │ o │ e │ u │ i │ d │ h │ t │ n │ s │ - │  ent │
 ├───────┴─┬─┴─┬─┴─┬─┴─┬─┴─┬─┴─┬─┴─┬─┴─┬─┴─┬─┴─┬─┴─┬─┴──────┤
 │ shift   │ ; │ q │ j │ k │ x │ b │ m │ w │ v │ z │  shift │
 └─────────┴───┴───┴───┴───┴───┴───┴───┴───┴───┴───┴────────┘
```
