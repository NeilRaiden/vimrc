## Notes:
- in INSERT mode "CTRL+k {char}=" enters Cyrilic symbols:
	- <c-k>d= output д (letted D), etc.
	- <c-k>D= output Д (capital D), etc.
	- <c-k>=P output ₽ (Rubel)
- see `:help dig` and `:digraphs!`
- kbd def: <https://kbdlayout.info/KBDRU/>
